org.apache.spark.sql.json

JsonTable

object JsonTable extends Logging with Serializable

:: Experimental :: Converts a JSON file to a SparkSQL logical query plan. This implementation is only designed to work on JSON files that have mostly uniform schema. The conversion suffers from the following limitation:

  • The data is optionally sampled to determine all of the possible fields. Any fields that do not appear in this sample will not be included in the final output.
